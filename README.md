# Leet Code

릿코드 알고리즘 문제

## 참여방법

1. 깃 클론하기
   git clone https://github.com/eagerithm/leetcode.git
2. 본인 이름이 없는 경우 깃헙아이디를 폴더명으로 지정해 추가하시면 됩니다.
3. 커밋 시 브랜치 따기 - 브랜치 명은 본인이름+한국식날짜 (예: liha-210513)
   git checkout -b "liha-210513"
4. 커밋 완료 후 풀리퀘 잊지 않기 😉
5. Don't you dare give up!
